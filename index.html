I switched to the markdown editor Mou. “Mou The missing Markdown editor for web developers”. A brilliant editor that shows your markdown text on the left-hand side and the actual wysiwyg on the right.

Furthermore I switched to using GitHub for Mac. As they say themselves:

At GitHub, we think that sharing code should be as simple as possible.
